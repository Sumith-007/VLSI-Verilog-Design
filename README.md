# VLSI Verilog Design

This repository contains Verilog HDL implementations and simulation testbenches developed while learning RTL Design and Digital Electronics.

## Modules

- ALU
- 2:1 Multiplexer
- 4:1 Multiplexer
- Decoder
- Encoder
- Priority Encoder
- Comparator
- Half Adder
- Full Adder
- Ripple Carry Adder (4-bit)
- Majority Circuit
- Parity Generator
- Counter

## Tools

- Verilog HDL
- Icarus Verilog
- GTKWave
- Visual Studio Code
